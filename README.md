# curaMind-AI

A HIPAA-compliant telehealth and diagnostic platform.

### Tech Stack
- **Backend**: Django & Django REST Framework
- **Frontend**: React (Vite)
- **Database**: SQLite (Development)

### Features
- Patient and Doctor role-based authentication.
- AI-driven symptom analysis with severity checking.
- Doctor dashboard for reviewing and confirming AI predictions.
- Patient dashboard for submitting symptoms and viewing history.
